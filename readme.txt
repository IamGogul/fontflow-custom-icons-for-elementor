= How to install FontFlow Plugin =
1.Login to Your WordPress Dashboard
2.Navigate to the Plugins Section
3.Choose "Add New"
4.Search for the Plugin: ( In the "Search plugins" box, type "fontflow" )
5.Select the fontflow Plugin & Activate it.

= Installing a WordPress plugin via FTP (File Transfer Protocol) involves uploading the plugin files to your server manually. =
1.Download the FontFlow Custom Icons for Elementor Plugin
2.Unzip the downloaded file. You'll typically get a folder containing all the plugin files.
3.Use an FTP client like FileZilla to connect to your server.
4.Once connected, navigate to the /wp-content/plugins/ directory in your WordPress installation on the server.
5.From your local machine, locate the folder extracted in step 2. Drag this folder into the /wp-content/plugins/ directory on your server.
6.Allow FileZilla or your FTP client to complete the upload. This might take a few moments, depending on the plugin's size and your internet speed.
7.Once the upload is complete, go to your WordPress dashboard. Navigate to the "Plugins" section. You should see the plugin you just uploaded listed there. Click "Activate" to activate the plugin.

= How to use =
1.First, install and activate the [Elementor Page Builder Plugin.](https://wordpress.org/plugins/elementor/)This plugin is necessary to use FontFlow Custom Icons Plugin.
2.Next, install & activate the [FontFlow Custom Icons for Elementor Plugin ](https://wordpress.org/plugins/fontflow-custom-icons-for-elementor)
3.Once both plugins are activated, you can access the FontFlow icon library within Elementor. When editing a page with Elementor, navigate to any Elementor widget that has an "Icons Control" in its settings.
4.Within the widget settings, look for the "Icons control". Click on it to open the icon library. Here, you can browse and select the icons you want to use.

== Frequently Asked Questions ==
= What is FontFlow Custom Icons for Elementor Plugin? =
FontFlow Custom Icons for Elementor is a WordPress plugin that enhances the Elementor Page Builder by providing a rich library of custom icons to use in your designs.
= How do I install FontFlow Custom Icons for Elementor Plugin? =
1.Log in to your WordPress Dashboard.
2.Navigate to Plugins > Add New.
3.Search for "FontFlow Custom Icons for Elementor".
4.Click Install Now and then Activate the plugin.
= Can I use FontFlow Custom Icons with any Elementor widget? =
Yes, you can use FontFlow Custom Icons with any Elementor widget that has an "Icons Control" in its settings.
= How do I access the FontFlow icon library? =
When editing a page with Elementor, go to a widget that has an "Icons Control" in its settings. Click on the "Icons Library" option to open the FontFlow icon library.
= Can I customize the icons? =
Absolutely! FontFlow allows you to customize the selected icons by adjusting their size, color, and other properties to match your design requirements.
= Are new icons regularly added to the library? =
Yes, we regularly update the FontFlow icon library, adding new icons to provide you with a diverse selection for your projects.
= Is FontFlow Custom Icons for Elementor Plugin compatible with the latest version of WordPress? =
Yes, FontFlow Custom Icons for Elementor is developed to be compatible with the latest version of WordPress as well as the current version of Elementor.
= Is there a premium version with additional features? =
As of now, FontFlow Custom Icons for Elementor is a free plugin, and all its features are available in the free version.
